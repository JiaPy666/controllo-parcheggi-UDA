# UDA
## Progetto finale: un piccola sistema distribuito a gestire i pacheggi 
- Un documento per gestire il progetto 
- Analisi dei requisiti del progetto scritto in md
- Un mappa per visualizzare
- Può scegliere il luogo disponibile(comune, città, via, ecc.)
- Può prenotare il posto per parcheggi
- Parcheggi disponibile per persone speciali

## Architettura
- Database: Mysql 
- Web app: react + js
- Script per gestire con db e web app: python + flask
